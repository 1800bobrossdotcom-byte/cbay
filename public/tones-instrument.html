MIDI & DAW INTEGRATION

Notes C2–F4 → Pads | CC1 Detune | CC7 Volume | CC91 Reverb
ROUTE TO DAW (VST-style)
Windows: Install VB-Cable (free) → set browser audio output to VB-Cable → set DAW input to VB-Cable. Use loopMIDI for virtual MIDI routing.
macOS: Install BlackHole (free) → set browser output to BlackHole → set DAW input to BlackHole. Use built-in IAC Driver for MIDI.
Linux: Use PulseAudio/PipeWire loopback module for audio routing.
Compatible with: FL Studio, Ableton, Logic Pro, Reason, Bitwig, Reaper, Pro Tools

STANDALONE APP (Electron)
npm init -y && npm i electron --save-dev
Create main.js loading this URL in a BrowserWindow → npx electron .
Electron exposes system audio I/O — ASIO on Windows, CoreAudio on Mac. Package with electron-builder for .exe/.dmg/.AppImage.

WEB AUDIO MODULE (WAM)
This instrument’s AudioWorklet architecture is WAM-compatible. Wrap as a WAM plugin to load inside WAM-enabled DAWs (e.g. webaudiomodules.org).
