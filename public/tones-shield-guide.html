1.1 DETECTION ENGINE

Real-Time FFT Frequency Analysis

When you activate the shield, your device's microphone captures ambient audio. The system runs a Fast Fourier Transform (FFT) with 8192 bins at 48kHz sample rate, giving approximately 5.86 Hz resolution per bin. This means it can distinguish frequencies with extreme precision.

The spectrogram waterfall display visualizes the entire audible spectrum in real-time. Each vertical column represents a moment in time; color intensity maps to amplitude (volume) at that frequency.

What it detects:

  • LRAD PRIMARY (2,000 - 3,500 Hz): The main LRAD frequency band. This is where Long Range Acoustic Devices concentrate their energy - the frequencies most painful to human ears.

The system classifies each detected threat by severity (HIGH/MED) based on frequency range and amplitude. Threats above -35 dB relative threshold trigger detection alerts.

1.2 ACTIVE PHASE CANCELLATION (ARC)

The Core Defense Mechanism

Phase cancellation is real physics, not theory. Sound is a pressure wave. If you generate the exact same wave, 180 degrees out of phase (inverted), the peaks of one wave fill the troughs of the other. They cancel. This is how noise-cancelling headphones work - the ARC Shield applies the same principle as an outward-facing shield.

AUTO-CANCEL: Tracks the dominant detected threat frequency in real-time. Generates a matching tone and routes it through a delay node calibrated to 1/(2 * frequency) seconds - which produces exact 180-degree phase inversion. As the incoming frequency shifts, the system tracks it with ~20ms response time.

SWEEP JAM: Generates a sawtooth wave that sweeps continuously from 1,500 Hz to 4,000 Hz and back over 1.6-second cycles. This creates destructive interference across the entire LRAD primary band. Even if you don't know the exact attack frequency, the sweep will cross it twice per cycle.

BROADBAND: White noise filtered through a bandpass centered at 2,500 Hz (Q=0.8). This floods the LRAD frequency band with random energy, disrupting the coherent beam pattern that gives LRAD its directional effectiveness.

PULSE COUNTER: Square wave at 2,500 Hz modulated by an 8 Hz amplitude pulse. The abrupt on/off pattern creates complex harmonic interference that disrupts the steady-state pressure wave of LRAD systems.

3.4 TROUBLESHOOTING: MICROPHONE ACCESS

Desktop Browsers (Chrome / Edge / Brave)

Desktop browsers enforce microphone permissions more strictly than mobile browsers. If the ARC Shield is not prompting for mic access on desktop:

  1. HTTPS Required: Microphone access is only available over HTTPS. Make sure you are accessing cbay.ing, not an insecure URL.
  2. Check Site Permissions: Click the lock/tune icon in the address bar > Site Settings > Microphone. Set to Allow.
  3. Clear Blocked State: If you previously denied mic access, the browser remembers. Go to Settings > Privacy & Security > Site Settings > Microphone. Find cbay.ing and remove it from the Blocked list, then reload.
  4. Browser Permissions-Policy: The server must send the correct Permissions-Policy: microphone=(self) header. This was fixed in the latest deployment - if you still have issues, hard-refresh with Ctrl+Shift+R.
  5. System Microphone: Ensure your OS allows the browser to access the microphone (Windows: Settings > Privacy > Microphone > allow desktop apps).

Mobile Browsers

Mobile browsers (especially Chrome on Android and Safari on iOS) are generally more lenient with microphone prompting. If mic access still fails:

  1. iOS Safari: Go to Settings > Safari > Microphone > set to Ask or Allow
  2. Android Chrome: Site Settings > Microphone > ensure cbay.ing is allowed
  3. PWA: If installed as a PWA, microphone permission is set at install time. Uninstall and reinstall if needed.
